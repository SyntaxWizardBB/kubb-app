# Design assets

Master vector files for the Kubb club brand mark and wordmark.

- `logo-mark.svg` — square app/badge mark (Meadow Badge, Variante A)
- `logo-wordmark.svg` — horizontal wordmark for headers

## Exporting app icons

`tools/export_icons.sh` rasterises `logo-mark.svg` into every platform size
(iOS asset catalog, Android mipmaps, Web/PWA, favicon, Apple touch icon) and
overwrites the existing PNGs in place. It is idempotent — re-running yields
identical files.

Requires one SVG renderer (first match wins):
`rsvg-convert` (`apt install librsvg2-bin` / `brew install librsvg`),
`inkscape`, or ImageMagick (`magick`/`convert`).

Run from repo root:

```bash
bash tools/export_icons.sh
```

Output paths are documented in the script header. The Android adaptive icon
foreground/background live as vector drawables under
`android/app/src/main/res/drawable/` and are not regenerated by the script.
